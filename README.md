# Udemy_clone
I have created a Udemy-style first page using HTML and CSS 💻 to replicate the look and feel of the popular online learning platform 🎓. The page features a responsive navigation bar 📱💡 that adapts seamlessly to different screen sizes, ensuring a user-friendly experience across devices. The navbar includes a prominent logo on the left 🏷️, closely resembling Udemy’s branding style, along with a centered search bar 🔍 designed for easy course discovery. The layout is clean and modern ✨, with attention to spacing, typography, and color scheme 🎨 to match the professional aesthetic of Udemy's interface.
